# zsh-tool Plugin

Use `mcp__zsh-tool__zsh` for zsh command execution with NEVERHANG protection and A.L.A.N. learning.

Prefer this over Bash when zsh-specific features are needed or when you want temporal learning about command patterns.

IMPORTANT: Tool output is NOT visible to the user. You MUST relay the command output in your response. Show the output verbatim — do not summarize, paraphrase, or omit. The user cannot see tool results directly.
